Actividad 1 
Esta actividad consiste en practicar programación web usando HTML, CSS y JavaScript. Se hicieron 10 ejercicios, cada uno enfocado en comprender lo básico de estructura de HTML, diseño con CSS y lógica con JavaScript

Los ejercicios realizados son los siguientes:
1. Hola Mundo – Impresión de la cadena Hola Mundo con la estructura básica de HTML.
2. Texto – Manejo de párrafos, encabezados y formato de texto.
3. Enlaces – Creación y uso de hipervínculos.
4. Imágenes – Manejo de imágenes en una página.
5. Integración De Imágenes E Hipervínculos – Combinar imágenes e hipervínculos.
6. Secciones – Organización de contenido usando etiquetas.
7. Receta: Tacos al Pastor – Página aplicando listas y estructura de contenido.
8. Horario – Uso de tablas para mostrar información organizada.
9. Contacto – Formulario de contacto con estilo minimalista.
10. Conversor de Temperatura – Calculadora que convierte grados de Celsius a Fahrenheit.

Tecnologías usadas:

- HTML para la estructura del contenido.
- CSS para darle estilo y diseño a cada página.
- JavaScript para agregar lógica.
